The code that powers [GMHost](http://gmhost.cryto.net/).

**WARNING:** Extremely messy, bad, and hacky code. Use at your own risk.

Requirements: 

* HTTPd (preferably something non-Apache)
* PHP 5.3+
* PHP-cURL
* Tahoe-LAFS
